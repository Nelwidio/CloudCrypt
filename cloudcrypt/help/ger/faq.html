FAQ

Was ist CloudCrypt?

CloudCrypt ist eine Software, welche es einem Nutzer ermöglicht, seine Daten verschlüsselt in einer Cloud abzulegen.

Unter welcher Lizenz steht die Software?

CloudCrypt steht unter der Lesser General Public License Version 3 (LGPLv3).

Warum wird eine Open-Source-Lizenz genutzt?

Die durchführenden Studenten des Projektes sind Anhänger der Open-Source-Bewegung. Uns ist es wichtig, dass der Nutzer auch wirklich die gesamte Kontrolle über die Software hat, die er einsetzt. Dazu gehört essentiell, dass der Source-Code des Programms offenliegt und jeder das Recht hat, diesen zu verbessern oder seinen Anforderungen anzupassen. Ein häufiges Missverständnis ist die Ansicht, dass freie Software immer kostenlos sein müsse. Nach Dr. Richard Stallman (einer der Begründer der Open-Source-Bewegung) entspricht das frei in freie Software vielmehr dem frei in Freiheit, als jenem in Freibier. Wir haben uns aber gegen eine entgeltliche Nutzung der Software entschieden.

Wie werden die Daten verschlüsselt?

Alle Daten, die in die Cloud abgelegt werden, werden mittels dem Advanced Encryption Standard (AES) mit einer Schlüssellänge von 256 Bits verschlüsselt. Hierbei greifen wir auf die Implementierung im GNU Privacy Guard zurück.

Welche Voraussetzungen hat CloudCrypt?

Zur Nutzung von CloudCrypt benötigen sie die folgenden Komponenten:

  • Standard-PC bzw. Notebook
  • Microsoft Windows 7 oder höher oder beliebige Linux-Distribution
  • Python in der Version 3.x
  • PyQt5
  • GNU Privacy Guard
  • einen gütigen Account für Dropbox

Welche Cloud-Services werden unterstützt?

Im Moment wird nur Dropbox unterstützt, da dies die von Entwicklern eingesetzte Cloud ist. Jedoch sollten alle Cloud-Servies funktionieren, die einen lokalen Ordner als Mirror der eigenen Cloud-Ordner verwenden.
